# 쿠팡 썸네일 다운로더 (크롬 확장)

쿠팡 상품 페이지에서 썸네일 후보를 뽑아 **원하는 1개를 다운로드**하는 팀 내부용 크롬 확장.
서버를 거치지 않고 여러분의 브라우저에서 동작하므로 **무료이고 쿠팡 차단에 걸리지 않습니다.**
MIT 라이선스 · 우리 팀이 소유·유지합니다.

## 쓰는 법 — 웹앱 안에서 (권장)
설치만 해두면, 그 뒤로는 **웹앱(썸네일 탭)에 쿠팡 링크를 붙여넣기만** 하면 됩니다.
확장 아이콘을 직접 누를 필요가 없습니다 — 웹앱이 뒤에서 확장을 호출해 이미지를 가져옵니다
(쿠팡 페이지가 잠깐 백그라운드 탭으로 열렸다 닫히는 게 정상입니다. 쿠팡 차단 우회에 실제 브라우저 탭이 필요하기 때문).

1. 웹앱의 **2. 썸네일 다운로드** 탭을 연다.
2. 쿠팡 상품 링크를 입력칸에 붙여넣고 **가져오기** 클릭.
3. 몇 초 뒤 후보 이미지들이 같은 화면에 뜬다.
4. 원하는 이미지의 **⬇ 다운로드** 클릭 → 저장 완료.

## 쓰는 법 — 확장 자체에서 (보조)
쿠팡 페이지를 직접 보고 있을 때는 확장 아이콘을 눌러도 동일하게 동작합니다.
1. 쿠팡 상품 페이지를 연다.
2. 브라우저 오른쪽 위 확장 아이콘에서 **쿠팡 썸네일 다운로더** 클릭.
3. **썸네일 가져오기** 버튼을 누른다 → 후보 이미지들이 뜬다.
4. 원하는 이미지의 **⬇ 다운로드** 클릭.

> 확장을 설치하기 *전에* 열어둔 쿠팡 탭이면 한 번 새로고침한 뒤 실행하세요.

## 설치 방법 A — 개발자 모드로 로드 (무료, 지금 바로)
1. 이 `extension/` 폴더를 팀원 PC에 복사(또는 압축 전달).
2. 크롬 주소창에 `chrome://extensions` 입력.
3. 오른쪽 위 **개발자 모드** 켜기.
4. **압축해제된 확장 프로그램을 로드합니다** → 이 `extension/` 폴더 선택.
5. 끝. (아이콘이 안 보이면 툴바의 퍼즐 조각에서 고정)

## 설치 방법 B — 크롬 웹스토어 비공개(unlisted) 배포 (권장, 팀 규모 클 때)
1. [Chrome 개발자 대시보드](https://chrome.google.com/webstore/devconsole) 가입(최초 1회 $5).
2. `extension/` 폴더를 zip으로 압축해 업로드, **공개 범위를 '비공개(unlisted)'** 로 설정.
3. 검수 통과 후 나오는 설치 링크를 팀에 공유 → 링크 눌러 설치(개발자 모드 불필요).
4. 그 링크를 웹앱의 `EXTENSION_INSTALL_URL` 에 넣으면 앱 '썸네일' 탭에 설치 버튼이 뜬다.

## 요구 권한 (왜 필요한지)
| 권한 | 이유 |
|---|---|
| `host_permissions: *.coupang.com` | 쿠팡 페이지에서 이미지 후보를 읽기 위함 |
| `downloads` | 고른 이미지를 저장하기 위함 |
| `tabs` | 웹앱 요청으로 쿠팡 페이지를 백그라운드 탭에서 열고 닫기 위함 |
| `activeTab`, `scripting` | 확장 아이콘으로 직접 쓸 때 현재 탭에 추출 스크립트를 실행하기 위함 |
| `externally_connectable` | 우리 웹앱 주소에서만 확장을 호출할 수 있도록 제한(다른 사이트는 호출 불가) |

수집한 이미지나 정보를 외부 서버로 전송하지 않는다.

## ⚠️ 배포 주소가 바뀌면 해야 할 일
`manifest.json`의 `externally_connectable.matches` 에 웹앱 주소가 등록되어 있어야 웹앱이 확장을 호출할 수 있습니다.
기본값은 로컬(`http://localhost/*`)과 `*.streamlit.app` 전체를 허용해 두었습니다.
**다른 도메인**(자체 도메인 연결 등)으로 배포하면, 그 주소를 `matches` 배열에 추가하고 팀원에게 확장을 다시 로드/재설치하도록 안내하세요.

## 확장 ID는 고정되어 있습니다
`manifest.json`의 `key` 필드 덕분에 어느 PC에 설치하든 확장 ID가 항상 `algnnfjoiiepjinfalghfnmmpjedehdg` 로 동일합니다.
(ID가 랜덤이면 웹앱이 어느 확장에 말을 걸어야 할지 알 수 없어 이렇게 고정했습니다.) 이 필드를 지우거나 바꾸지 마세요.

## 파일
- `manifest.json` 확장 정의(MV3), 고정 ID, 웹앱 허용 목록
- `background.js` 웹앱의 요청(추출/다운로드)을 받아 처리하는 서비스워커
- `content.js` 쿠팡 페이지에서 이미지 후보 추출
- `popup.html` / `popup.js` 확장 아이콘 클릭 시 보조 화면(후보 표시·다운로드)
