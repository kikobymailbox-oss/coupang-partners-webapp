# 쿠팡 썸네일 다운로더 (크롬 확장)

쿠팡 상품 페이지에서 썸네일 후보를 뽑아 **원하는 1개를 다운로드**하는 팀 내부용 크롬 확장.
서버를 거치지 않고 여러분의 브라우저에서 동작하므로 무료입니다.
MIT 라이선스 · 우리 팀이 소유·유지합니다.

## 쓰는 법
1. 쿠팡 상품 페이지를 **직접 연다** (웹앱의 "이 쿠팡 상품 페이지 열기" 버튼을 눌러도 됨).
2. 브라우저 툴바의 **쿠팡 썸네일 다운로더** 아이콘 클릭.
3. **썸네일 가져오기** 버튼을 누른다 → 후보 이미지들이 뜬다.
4. 원하는 이미지의 **⬇ 다운로드** 클릭.

> ⚠️ **꼭 쿠팡 페이지를 사람이 직접 열고 나서** 확장을 눌러야 합니다. 예전에는 웹앱이
> 프로그램으로 쿠팡 탭을 자동으로 열어주는 기능이 있었으나, 실측 결과 **"프로그램이 여는
> 탭"만 쿠팡 봇 차단에 걸리고 "사람이 직접 열고 클릭"하는 방식은 항상 정상 동작**함이
> 확인되어 자동화 기능은 제거했다(2026-07-02, `../DECISIONS.md`).

## 설치 방법 A — 개발자 모드로 로드 (무료, 지금 바로)
1. 이 `extension/` 폴더를 팀원 PC에 복사(또는 압축 전달) — 웹앱의 설치 안내(사이드바 → 🧩 확장 설치 안내)에서 zip으로 바로 받을 수 있음.
2. 크롬 주소창에 `chrome://extensions` 입력.
3. 오른쪽 위 **개발자 모드** 켜기.
4. **압축해제된 확장 프로그램을 로드합니다** → 압축 해제한 `coupang-thumbnail-extension` 폴더 선택.
5. 끝. (아이콘이 안 보이면 툴바의 퍼즐 조각에서 고정)

## 설치 방법 B — 크롬 웹스토어 비공개(unlisted) 배포 (권장, 팀 규모 클 때)
1. [Chrome 개발자 대시보드](https://chrome.google.com/webstore/devconsole) 가입(최초 1회 $5).
2. `extension/` 폴더를 zip으로 압축해 업로드, **공개 범위를 '비공개(unlisted)'** 로 설정.
3. 검수 통과 후 나오는 설치 링크를 팀에 공유 → 링크 눌러 설치(개발자 모드 불필요).
4. 그 링크를 웹앱의 `EXTENSION_INSTALL_URL` 에 넣으면 설치 안내에 설치 버튼이 뜬다.

## 요구 권한 (왜 필요한지)
| 권한 | 이유 |
|---|---|
| `host_permissions: *.coupang.com` | 쿠팡 페이지에서 이미지 후보를 읽기 위함 |
| `downloads` | 고른 이미지를 저장하기 위함 |
| `activeTab`, `scripting` | 지금 보고 있는 탭에 추출 스크립트를 실행하기 위함 |
| `externally_connectable` | 웹앱이 "확장이 설치돼 있는지"만 확인(PING)할 수 있도록 — 우리 웹앱 주소에서만 호출 가능 |

수집한 이미지나 정보를 외부 서버로 전송하지 않는다.

## 확장 ID는 고정되어 있습니다
`manifest.json`의 `key` 필드 덕분에 어느 PC에 설치하든 확장 ID가 항상 `algnnfjoiiepjinfalghfnmmpjedehdg` 로 동일합니다.
(웹앱이 "확장 설치됨" 여부를 확인(PING)할 때 이 ID로 특정 확장에만 말을 겁니다.) 이 필드를 지우거나 바꾸지 마세요.

## 파일
- `manifest.json` 확장 정의(MV3), 고정 ID, 웹앱 허용 목록
- `background.js` 웹앱의 설치 확인(PING) 요청에만 응답하는 서비스워커 (추출/다운로드는 관여 안 함)
- `content.js` 쿠팡 페이지에서 이미지 후보 추출
- `popup.html` / `popup.js` 확장 아이콘 클릭 시 화면(후보 표시·다운로드) — 실제 사용 흐름의 전부
